# Product Dashboard

A browser-based product management dashboard built with vanilla JavaScript, HTML, and CSS — no frameworks or libraries.

## Features
- View and manage a list of products
- Dynamic DOM manipulation using plain JavaScript
- Clean, responsive layout styled with custom CSS

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

## How to Run
1. Clone the repository
2. Open `index.html` in your browser — no build step required

## What I Learned
This project reinforced my understanding of core JavaScript concepts including DOM manipulation, event handling, and working with arrays and objects — all without relying on a framework.
